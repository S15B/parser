# HTML Image Parser

Этот проект содержит реализацию **HTML-парсера на Python**, предназначенного для поиска изображений внутри HTML-документа.

Парсер анализирует HTML-файл, извлекает ссылки на изображения из различных элементов страницы, сохраняет найденные изображения и формирует JSON-файл с результатами и контекстом.

Особенность проекта — **собственная реализация HTML-парсера без сторонних библиотек** (без `BeautifulSoup`, `lxml` и т.д.).

---

# Возможности

Парсер умеет находить изображения в различных местах HTML:

### HTML-теги

* `<img src="">`
* `<img srcset="">`
* `<image href="">` (SVG)

### Дополнительные атрибуты

Поддерживаются различные атрибуты, которые часто используются для ленивой загрузки изображений:

* `data-src`
* `data-lazy`
* `data-original`
* `data-image`
* `data-bg`
* `data-background`
* `poster`
* `background`

### CSS

Изображения извлекаются из:

* `style="background-image: url(...)"`

а также из CSS внутри:

```
<style>
background-image: url(...)
</style>
```

### Meta-теги

Поддерживаются изображения из метаданных страницы, например:

```
<meta property="og:image">
```

### Link-теги

Например:

```
<link rel="preload" as="image">
```

# Поддерживаемые источники изображений

Парсер может работать с изображениями из:

* HTTP / HTTPS ссылок
* относительных путей
* локальных файлов
* Data URL (`data:image/...`)

---

# Результат работы

После выполнения создаются:

* папка с сохранёнными изображениями
* файл `results.json` с информацией о них

Структура:

```
output/
├── image_1.jpg
├── image_2.png
├── image_3.webp
└── results.json
```

---

# Формат JSON

Пример записи:

```json
{
  "id": 1,
  "image_name": "image_1.jpg",
  "image_path": "/path/to/output/image_1.jpg",
  "original_src": "https://example.com/image.jpg",
  "source_type": "img_src",
  "alt": "Описание изображения",
  "text_before": "Текст перед картинкой",
  "text_after": "Текст после картинки",
  "context": "контекст вокруг изображения"
}
```

Контекст формируется из:

* текста перед изображением
* `alt`
* текста после изображения

# Установка

Требования:

* Python **3.8+**

Проект не использует внешние зависимости и работает только на стандартной библиотеке Python.

---

# Использование

## Базовый запуск

```
python html_parser.py page.html
```

## Указание папки для результатов

```
python html_parser.py page.html output/
```

## Использование base URL

Если HTML содержит относительные ссылки:

```
python html_parser.py page.html output/ https://example.com
```

# Особенности реализации

* собственный HTML-парсер
* работа без сторонних библиотек
* поддержка разных кодировок HTML:

  * UTF-8
